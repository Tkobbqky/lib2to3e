　　足球一直是，也将永远是我的生命与事业。这项运动让我的人生变得丰富多彩，带给我许多值得珍惜的挚友。足球需要永无止境的学习与探索，更需要在球场上永不停歇的实践。
n
　　在我的本书《我更爱奖杯》中，我几乎完全是从个人情感的角度来讲述我的教练生涯。而这本书的目标读者既包括足球教练，也包括广大足球爱好者。
n
　　首先，本书的经验对我的同行来说具有普适性。其次，足球爱好者尽管不需要真的成为一名教练，但仍可在我的书中找到足球比赛的技术与策略，如训练计划的制定、重要赛事的管理，以及与球员、俱乐部、球迷及媒体的关系处理等。
n
　　我想讲述我的心路历程，想告诉你们我是怎么理解教练这个角色的。教练并不仅仅是制定战术与比赛模式的人。想成为一名好教练，需要终生学习。每个教练都有自己喜欢的战术体系，但一名认真且有素养的职业教练需要了解并能随时运用所有其他战术体系。刚开始我喜欢用4-4-2阵型，但随后便发现该阵型与实际情况发生冲突，时间与经验是战术的佳打磨剂。好的战术就像一件量体裁制的衣服，教练将这件衣服穿在球员（常为冠军球员）身上，让他们能充分发挥自己的实力。好的战术同时也应能很好地表达出一家俱乐部的文化与历史背景。
n
　　在当今社会，制定比赛的战术不仅需要思考技术层面，还应考虑经济与市场需求。教练应统筹大局，制定出有效而具体的比赛模式，充分利用每个球员的才能。
n
　　然而，我不希望读者将此书看作一本足球比赛手册。我再次重申，教练不只是要将一支队伍训练到佳状态。我无疑会谈及我所使用的战术、策略与模式，但我更愿意与读者分享我过去与现在工作中的一些重要时刻，分享那些鲜为人知却同样重要的经验。

作者简介

[

安切洛蒂，1959年出生于意大利雷焦艾米利亚大区的雷乔罗镇。他先后效力于帕尔马、罗马与AC米兰。退役后曾出任意大利国家队助教，与教练阿里戈 萨基共事。他执教过雷吉纳、帕尔马、尤文图斯、AC米兰、切尔西与巴黎圣日尔曼。2013年夏天，他开始执教皇家马德里。2009年，他与亚历山大阿尔恰多合作出版了《我更爱意大利杯》（里佐利出版社）。安切洛蒂拿到了意甲、英超、法甲、德甲四大联赛的冠军。
