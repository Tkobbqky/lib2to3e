内容简介

本书总结了国内外各类情报体系的架构与组成, 分析了突发事件应急决策的快速响应对情报流的需求, 深入探讨了应急响应情报体系应具备功能, 融合多方面因素阐述了应急响应情报体系应具备如下特征: 具有大数据环境下情报采集与处理能力 ; 具有资源保障和知识普及能力 ; 具有实现快速响应的技术保障能力 ; 具有应急决策的辅助支持能力 ; 具有扁平化的柔性组织架构 ; 具有对突发事件处理的评价能力。为了架构具备上述能力的应急响应情报体系, 该成果重点研究了: 应急响应情报体系架构 ; 应急响应情报采集与处理 ; 应急响应情报加工与组织 ; 应急响应情报分析与服务 ; 应急响应情报评估 ; 并给出了智慧城市的情报体系架构, 同时还针对已经发生的突发事件, 进行了情报体系融入其中的预警、处理和善后工作的解读。
